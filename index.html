How to connect
1
Peer A: Click Create Offer and share the offer code with Peer B.
2
Peer B: Paste the offer code into the Offer box and click Generate Answer. Share the answer code back to Peer A.
3
Peer A: Paste the answer. Click Connect. Status changes to Connected. Files, chat and voice are all ready.

⚠ Both peers must keep this tab open. For cross-network use, TURN is used automatically. For large files (>500 MB) use Chrome — it streams directly to disk without loading into RAM.
