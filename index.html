📡 PeerDrop 95 — P2P File & Chat Transfer
How to connect
1
Peer A: Click Create Offer. Copy the code from the Offer box and send it to Peer B (email, chat, etc.).
2
Peer B: Paste the code into the Offer box. Click Generate Answer. Copy the answer and send it back to Peer A.
3
Peer A: Paste the answer into the Answer box. Click Connect. Status will change to Connected.

⚠ Both peers must have this page open at the same time. For cross-network use, a TURN relay server may be required.
